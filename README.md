# Bunken: A reference manager for researchers

Bunken は、研究者向けの軽量な文献管理ツールです。  
*自己責任での利用をお願いします。

## Features

- DOI / arXiv から文献情報を取得
- BibTeX の読み込み
- PDF からメタデータを抽出
- PDFやsupplementary files を自動整理して保存
- タグ・コレクションで文献を整理
- 読書ステータスを管理
- メモを保存
- BibTeX を出力
- 指定した保存先フォルダを使って文献とPDFを管理
- Dropbox や Google Drive などで同期されるフォルダを保存先にすれば、複数のデバイスから同じライブラリを利用可能

## Quick Start

1. `bunken_v[year]-[version].html` をブラウザで開く
2. 必要に応じて「保存先フォルダを選択」から保存先を設定
3. 「＋追加」から DOI / arXiv / BibTeX / PDF を使って文献を登録

## Notes

- データはブラウザ内、または指定したローカルフォルダに保存されます
- サーバは不要です
- Dropbox や Google Drive などで同期しているフォルダを保存先に指定すると、複数デバイス間で同じデータを扱えます
- Chrome / Edge のみ対応（Safari は非対応）
